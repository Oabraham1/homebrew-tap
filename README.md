# Homebrew Tap

Custom Homebrew formulas by Ojima Abraham.

## Install

```bash
brew tap oabraham1/tap
brew install mcp-scanner
```

Or directly:

```bash
brew install oabraham1/tap/mcp-scanner
```

## Available Formulas

| Formula | Description |
|---------|-------------|
| mcp-scanner | Security scanner and proxy for MCP servers |
