# Homebrew Tap

Custom Homebrew formulas by Ojima Abraham.

## Install

```bash
brew tap oabraham1/tap
brew install mcp-guard
```

Or directly:

```bash
brew install oabraham1/tap/mcp-guard
```

## Available Formulas

| Formula | Description |
|---------|-------------|
| mcp-guard | Security scanner and proxy for MCP servers |
